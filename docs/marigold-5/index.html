1. Device Overview

The Marigold-5 is a synchronous step-down (buck) DC-DC converter module designed to provide a regulated 5.1V rail for FRC robotics applications. It accepts a wide input voltage range (7V‚Äì18V) suitable for FRC lead-acid battery architecture and distributes power via four USB-C receptacles.

Built around the Texas Instruments TPS56A37, it solves the common headache of powering coprocessors (Raspberry Pi, Orange Pi) and vision cameras by providing robust, high-current 5V power with secure mounting.

Thermal Design

At full 10A load, the regulator dissipates approx. 3-4W. The PCB has an aluminum heatsink block bonded to the bottom side. When bolted to a 1x2" aluminum tube, the robot frame acts as a thermal reservoir, eliminating the need for fans.

5. Pinouts & Wiring

5.1 Input Power (VIN)

Connector: #6-32 Ring Terminal Studs

  • RED (Side A): +12V Input (Battery)
  • BLACK (Side B): GND

5.2 Auxiliary Output

Connector: #4-40 Ring Terminal Studs

  • RED: +5.1V Output
  • BLACK: GND

Usage: For powering 5V fans, LEDs, or logic shifters. Shares the 10A total limit.
